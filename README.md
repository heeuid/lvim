# lvim
my lunarvim setting

## Structure
- lvim
    + ~/.config/lvim
- lvim.sh
    + lvim.sh build
    + lvim.sh config

## Usage
1. install neovim & lunarvim
    + lvim.sh build 
2. configure lvim
    + lvim.sh config
3. open lunarvim
    + lvim
4. key bindings
    + open lvim/lua/settings/which_key.lua
    + "<space>[word]"
    + or "\[words]"
